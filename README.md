# HR–IT Employee Onboarding Automation Copilot

## Project Overview
AI assistant built using Microsoft Copilot Studio to automate HR–IT onboarding tracking.

## Features
- Generate onboarding summaries for managers
- Track employee onboarding status
- Answer HR/IT onboarding queries
- Summarize onboarding-related issues
- Responsible AI compliant

## Technologies Used
- Microsoft Copilot Studio
- Azure AI Infrastructure (managed)
- Knowledge-based dataset grounding

## Dataset
The assistant uses a structured onboarding dataset to retrieve:
- Department
- Role
- Required IT Assets
- Onboarding Status

## Responsible AI
- No hallucination
- No data modification
- Dataset-grounded responses
- Neutral and unbiased outputs
